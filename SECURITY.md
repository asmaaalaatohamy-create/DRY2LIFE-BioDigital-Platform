# Security Policy - DRY2LIFE

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it immediately.

### How to Report

1. **Do not open a public Issue** - this could put users at risk
2. Send an email to: **asmaaalaatohamy@gmail.com**
3. Provide as much information as possible:
   - Type of vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fixes

### What Happens Next?

1. We will confirm receipt within 48 hours
2. We will investigate within 7 days
3. We will keep you updated on progress
4. We will release a security update if needed

## Secure Environment

We encourage all contributors to:
- Use strong passwords
- Do not share sensitive data
- Regularly update dependencies
