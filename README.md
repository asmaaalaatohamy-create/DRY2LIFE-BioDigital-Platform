## 📖 Overview

**DRY2LIFE** is a bio-digital intelligence platform specialized in **soil salinity management**, combining:

- 📡 **IoT sensors** for real-time monitoring
- 🧬 **PGPR-based bioremediation** (Plant Growth-Promoting Rhizobacteria)
- 🤖 **AI-powered decision engine** for precision agriculture
- 📊 **Validated Egyptian research** ensuring accuracy and effectiveness

---

## ✨ Key Features

### 🔬 Smart Monitoring
- Live sensor simulation with IoT integration
- 7-day trend analysis for salinity, moisture, and temperature
- Interactive dashboard with real-time updates

### 🧬 Bioremediation Module
- Custom PGPR recommendations based on salinity levels
- Zone classification (Low/Moderate/High Salinity)
- Yield prediction with and without PGPR treatment

### 💧 Smart Irrigation Management
- Seasonal water requirement calculations
- Precision irrigation scheduling for salinity control
- Priority-based irrigation alerts

### 🤖 Smart Decision Engine
- Automatic alerts for critical conditions
- Precision recommendations for irrigation and treatment
- 30-day performance improvement forecasts

## 🚀 Installation & Setup

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Installation Steps

```bash
# 1. Clone the repository
git clone https://github.com/your-username/DRY2LIFE-BioDigital-Platform.git
cd DRY2LIFE-BioDigital-Platform

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the application
streamlit run app.py
